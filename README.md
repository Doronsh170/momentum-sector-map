# מניות מומנטום לפי תתי־סקטורים

קובץ סטטי מוכן להעלאה ל-GitHub Pages.

## שימוש
1. העלה את `index.html` לשורש ה-Repository.
2. ודא ש-GitHub Pages מופעל על branch `main` ועל `/root`.
3. לאחר הפריסה, האתר ייטען ישירות מדף הבית של ה-Repository.

אין צורך בבילד, התקנות או קבצי תלות

גילוי נאות:
התוכן באתר מיועד לצורכי מידע, מעקב והשוואה יחסית בלבד. אין לראות בו ייעוץ השקעות, שיווק השקעות, המלצה לקנייה או מכירה של ניירות ערך, או תחליף לייעוץ המתחשב בצרכים ובנתונים האישיים של כל אדם. השימוש במידע הוא באחריות המשתמש בלבד.
.
